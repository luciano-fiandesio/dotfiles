# Git

Symlink gitconfig files.

The `.gitconfig` file requires a local `.gitconfig.local` file, where sensitive info are stored.

This is a template:

[user]
  name = [name lastname]
  email = [email]
  useconfigonly = true
  signingkey = [sign key] 
[github]
  user = [github user] 
  token = [github token]
